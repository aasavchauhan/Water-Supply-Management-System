### 1. Reporting Bugs
- Check the [Issues](https://github.com/aasavchauhan/WaterSupplyManagement/issues) tab to see if the bug has already been reported.
- If not, create a new issue using the **Bug Report** template.
- Provide clear steps to reproduce the issue.

### 2. Suggesting Features
- Use the **Feature Request** template to propose new ideas.
- Explain the "Why" and "How" of your feature clearly.

### 3. Code Contributions
1. **Fork** the repository.
2. Create a new branch: `git checkout -b feature/MyNewFeature`.
3. Make your changes and commit them: `git commit -m "Add some feature"`.
4. Push to your branch: `git push origin feature/MyNewFeature`.
5. Submit a **Pull Request (PR)**.
